01About

I started at JUIT in 2025. First semester was C and how memory works, second semester brought object oriented programming in C++, classes, inheritance, and the first data structures. This summer I went deeper into the web, learning HTML that carries meaning and CSS layouts that don’t fall apart the moment someone opens them on a phone.

This site is part of that practice. No frameworks and no template, just one HTML file and one stylesheet written by hand. Everything I make goes up on GitHub, partly for my university’s summer training and mostly because building in public keeps me honest.
